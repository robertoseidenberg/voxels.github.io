Usage notes

  • Voxels is designed decidedly for use with a pencil. It's actully impossible to draw without one
  • Touch interactions are solely used for navigating the menu palettes. The pencil is used solely for drawing
  • Drawing is pressure sensitive. When the pencil rests on the display the insertion point is beeing highlighted
  • Any drawing operation is beeing performed by applying a bit of pressure
  • It works like any other iOS drawing software but in 3D
  • Panels might be closed by swiping them towards the screen edges using two fingers
